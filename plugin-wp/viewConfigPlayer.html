Configurações do Player

Informe sua KeyPlayer: ✶
A KeyPlayer é criada pelo gerador de player.
Navegar com Transmissão: " /> Mantém o player ligado sem interrupções ao navegar entre páginas.
ID do Container Principal: ✶
Descubra o ID e informe.
O conteúdo de cada página no site fica dentro do container principal, ao navegar de uma página para a outra, tudo que fica dentro é atualizado.
Caso troque o tema do seu site no Wordpress, o ID do novo tema pode ser diferente ao do anterior, portanto é necessário descobrir o novo ID e atualiza-lo aqui.
Class do Container Menu: ✶
O menu também pode mudar sua exibição ao navegar entre as páginas, portanto descubra a classe pai do menu principal.
Efeito na Barra de Rolagem: " /> Leva o usuário até o ínicio do conteúdo de cada página ao navegar.
